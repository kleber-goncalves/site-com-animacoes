# FP Selection

Este projeto foi desenvolvido com fins educacionais, como parte do estudo de HTML, CSS e JavaScript.

## Objetivos

O objetivo deste projeto é criar uma página web responsiva que exibe informações sobre diferentes carros, incluindo imagens, descrições e detalhes.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Navegação entre as diferentes seções da página
- Exibição de imagens, descrições e detalhes dos carros
- Indicação visual do carrossel atualmente selecionado
- Transição suave entre as diferentes seções da página

## Como executar o projeto

1. Clone o repositório para o seu ambiente local.
2. Abra o arquivo `index.html` em seu navegador.
3. Experimente a página e navegue entre as diferentes seções.
